This project was developed as part of the SkillCraft Technology Web Development Internship.

## Features

- Start Timer
- Pause Timer
- Reset Timer
- Lap Time Recording
- Responsive User Interface
- Accurate Time Tracking

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Task Objective

Build an interactive and user-friendly stopwatch web application that allows users to accurately measure, record, and manage time intervals.
